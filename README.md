林中物语 — 2D 横版动作 RPG
一款快节奏的 2D 横版动作角色扮演游戏。挥剑斩敌、解锁技能、击败 Boss，在多张随机地图中一路推进到通关。

游戏类型
横版动作 RPG / 银河恶魔城类

操作方式
按键	功能
A / D	左右移动
空格	跳跃（可二段跳）
J	攻击（最多三段连击）
K	冲刺
L	掷剑
靠墙时按空格	墙跳
靠墙下落	墙滑
游戏流程
主菜单 → 第一关（森林） → 第二关（随机地图 ×3 选 1）→ 第三关（随机地图 ×3 选 1）→ Boss 关 → 通关
每关之间通过传送门衔接，进入传送门后有过场动画。中间两关每次进入都会随机切换地图布局，让重复游玩不会太单调。关卡中分布着金币和宝箱（普通宝箱 / 金宝箱），金币用于商店消费。

核心玩法
战斗 近战是主要输出手段。三段连击有递增的攻击距离和伤害，空中也能攻击。冲刺有无敌帧可以用来闪避，靠近墙壁时可以墙滑和墙跳到达高处平台。敌人攻击瞬间玩家可以触发反击——成功后敌人进入眩晕状态，趁机输出。

元素效果 攻击命中后会概率触发元素附加效果：

火焰：敌人持续掉血，头顶冒红字
冰霜：敌人移动和攻击速度大幅降低
雷电：充能叠加，满后落下一道雷击造成范围伤害
三种元素同一时间只能挂一个（雷电可叠加充能），需要根据敌人类型选择合适的技能搭配。

技能系统 游戏内有 4 个可解锁的主动技能：

技能	基础效果	进阶效果
冲刺	快速位移	冲刺路径上生成碎片爆炸
掷剑	远程投掷武器	附带火焰伤害
剑气	近距离爆发	附带冰霜减速
碎片	投掷碎片爆炸	附带瞬移回血
每个技能有三个升级阶段：激活 → 进阶（解锁元素附加）→ 强化（反复花费金币提升伤害倍率）。

商店 关卡中会遇到商店 NPC，靠近后按 E 打开商店。可以花金币进入角色属性升级界面或技能强化界面。角色属性包括攻击力、暴击率、最大生命值等，技能强化走上面说的三阶段路线。

敌人介绍
关卡中会遇到以下敌人：

骷髅 — 最基础的敌人，来回巡逻，发现玩家后追击并近身攻击。适合用来熟悉战斗手感。
哥布林 — 比骷髅多一段连击，第二段攻击距离更远，需要拉开距离或用冲刺躲开。
精灵弓箭手 — 远程敌人，发现玩家后会射箭。箭矢是投射物，需要跳跃或冲刺闪避。
蝴蝶 — 飞行敌人，会向玩家发射弹幕。死亡时会爆炸，注意拉开距离。
Boss 战：收割者
最终关卡的 Boss，分两个阶段：

一阶段（100%~50% 血量） Boss 在竞技场内来回巡逻，发现玩家后追击近战攻击，偶尔瞬移到随机位置。攻击节奏不算快，主要用来熟悉 Boss 的攻击前摇和闪避时机。

二阶段（50% 血量以下触发） 屏幕闪白，Boss 进入狂暴：

瞬移频率大幅提高
新增法术攻击——向玩家脚下投射落箭，需要持续移动躲避
获得生命回复能力，不能慢慢磨血
技能冷却时间减半
二阶段需要更积极地进攻，同时注意躲避地面法术区域。掌握反击时机是关键——成功反击可以让 Boss 眩晕，争取到宝贵的输出窗口。
